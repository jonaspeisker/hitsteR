# hitsteR

[R](r) functions to make a song guessing game. Based on a Spotify playlist, they generate a pdf with artist, year, and song title on one side and a QR code to play the song on the other side.

![Example output](output/example.png)

## How to play

The rules of game can be found [here](https://hitstergame.com/en-us/how-to-play/).
